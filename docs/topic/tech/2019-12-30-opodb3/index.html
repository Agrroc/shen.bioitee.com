从 Jupyter Notebook 升级到 JupyterLab

2019-12-30

JupyterLab is a next-generation web-based user interface for Project Jupyter.

安装

JupyterLab 可以通过 conda, pip, pipenv or docker 的方式安装。

JupyterLab 需要 4.3 版本以上的 notebook，如果 notebook 没有安装需要先安装 notebook。

conda install notebook
conda install jupyterlab

详细的安装，参考官方文档：https://jupyterlab.readthedocs.io/en/stable/getting_started/installation.html

启动

jupyter lab

详细启动方法，参考：https://jupyterlab.readthedocs.io/en/stable/getting_started/starting.html
